# 🌐 **KeenKeeper — Keep Your Friendships Alive**

## 📝 **Description**

A smart relationship management app that helps you stay connected with your friends by tracking interactions, setting communication goals, and providing insights so no important connection fades away.

---

## 🚀 **Live Demo**

🔗 [keen-keeper-site](https://kin-keeper-project-a7.vercel.app/)

---

## 🛠️ **Technologies Used**

* ⚛️ Next.js
* 🔀 App Router
* 🎨 Tailwind CSS
* 📊 Recharts

---

## ✨ **Key Features**

* 🧑‍🤝‍🧑 Track friends with real-time status (overdue, almost due, on-track)
* ⚡ Quick check-in system (Call, Text, Video) with instant timeline updates
* 📈 Visual analytics dashboard with Pie Chart insights
